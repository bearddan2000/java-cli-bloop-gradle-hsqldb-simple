# java-cli-bloop-gradle-hsqldb-simple

## Description
A java gradle build, that connects to hsqldb
database.

## Tech stack
- java
- bloop
- gradle
- hsqldb driver

## Docker stack
- gradle:jdk11-alpine

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
